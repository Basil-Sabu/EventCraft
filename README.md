# EventCraft – Event Management System

EventCraft is a full-stack **Spring Boot–based Event Management System** designed to manage events efficiently by handling clients, events, guests, vendors, expenses, and payments through well-structured admin and client portals.

This project demonstrates real-world backend and frontend development practices using Spring Boot and related technologies.

---

## Features

### Authentication & Roles
- Admin login
- Client login
- Session-based access control

### Admin Module
- Client management
- Event creation and management
- Guest group management 
- Vendor assignment and expense tracking
- Client payment tracking
- Reports and dashboards

### Client Portal
- View assigned events
- View guest details
- View vendor assignments
- View payment history and payment status

### Business Logic Highlights
- Normalized database design
- No redundant total/balance storage (computed dynamically)
- Duplicate prevention for vendor assignments
- Clean service-layer logic

---

## Tech Stack

- **Backend:** Java, Spring Boot, Spring MVC
- **Persistence:** Spring Data JPA, Hibernate
- **Frontend:** Thymeleaf, HTML, CSS
- **Database:** MySQL
- **Build Tool:** Maven
- **Version Control:** Git, GitHub
